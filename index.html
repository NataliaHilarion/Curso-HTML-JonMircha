Etiquetas Semanticas Estructurales

  • Header - Cabecera de un sitio web o de una seccion
  • Main - Define la sección principal del documento - Solo existe una por documento
  • Footer - Pie de página de un sitio web o de una sección
  • Nav - Representa una navegación
  • Article - Representa una sección de autocontenido (por si sola se explica)
  • Aside - Representa contenido complementario o secundario
  • Section - Representa una seccion de contenido genérico
  • Address - Representa una infomarción de contacto
