Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Utiliza una serie de elementos y etiquetas que describen el contenido y la estructura de una página, como encabezados, párrafos, enlaces, imágenes, y formularios.

Etiquetas de Bloque vs Etiquetas de Linea

En HTML, las etiquetas de bloque, como div, p y header, estructuran el contenido creando secciones que ocupan todo el ancho disponible y siempre comienzan en una nueva línea, organizando el layout de la página. Por otro lado, las etiquetas en línea, como span, a y em, se integran dentro del flujo del texto y sólo ocupan el espacio necesario, permitiendo estilizar partes específicas del contenido sin interrumpir el flujo

La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas
La etiqueta de linea por excelencia es la span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido Hola soy otra span

Estilos CSS en HTML

La etiqueta style en HTML se utiliza para definir estilos CSS internos directamente dentro de un documento HTML. Colocada generalmente dentro del head

  1. Directamente en la etiqueta de HTML con el atributo style
  2. Abrir estilos en el HEAD del HTML con la etiqueta Style{} y ahí directamente agregar estilos
  3. Enlazando una hoja de estilos con la etiqueta link rel="stylesheet" href="direccion de la .css">

Scripts en HTML

La etiqueta script en HTML se utiliza para incorporar y ejecutar código JavaScript en una página web. Puede contener scripts internos escritos directamente dentro de la etiqueta o vincular archivos JavaScript externos mediante el atributo src

  • Con la etiqueta script directamente en el HTML al final del Body
  • Con la etiqueta script pero con el atributo src

Imagenes

La etiqueta img en HTML se utiliza para incorporar imágenes en una página web. Es un elemento vacío, lo que significa que no tiene una etiqueta de cierre y se define mediante atributos como src source para especificar la URL de la imagen, y alt (alternate text) para proporcionar una descripción alternativa de la imagen

SVG - vectores

SVG Scalable Vector Graphics es un formato de imagen basado en XML que se utiliza para definir gráficos vectoriales bidimensionales en HTML. A diferencia de las imágenes rasterizadas, como JPEG o PNG, SVG puede escalarse a cualquier tamaño sin pérdida de calidad, lo que lo hace ideal para gráficos en la web que necesitan ser responsivos. Además, SVG permite la manipulación a través de CSS y JavaScript, facilitando la creación de gráficos interactivos y animaciones. Al ser un estándar abierto, es ampliamente compatible con la mayoría de los navegadores modernos.

Figuras

La etiqueta Figure en HTML se utiliza para encapsular contenido visual como imágenes, gráficos, diagramas o fragmentos de código, junto con su correspondiente leyenda. Esta leyenda se define mediante la etiqueta Figcaption

Listas

Las listas en HTML permiten organizar información en forma de ítems secuenciales o desordenados. Existen tres tipos principales de listas: las listas no ordenadas ul - Unorder list, que presentan los ítems con viñetas; las listas ordenadas ol - Order List, que enumeran los ítems con números; y las listas de definición dl - Definition List, que se usan para términos y sus descripciones. Los elementos individuales de una lista se definen con la etiqueta li en el caso de listas ordenadas y no ordenadas, y con dt (definition term) y dd (description data) en el caso de listas de definición. Estas estructuras son fundamentales para mejorar la organización y legibilidad del contenido en una página web.
